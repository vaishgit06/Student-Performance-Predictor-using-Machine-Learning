# Student-Performance-Predictor-using-Machine-Learning
This project uses Machine Learning to predict a student’s academic performance based on factors such as study hours, attendance, and previous marks. The aim is to identify patterns that influence learning outcomes and provide early insight into expected results.

## Description
This project predicts student marks using Machine Learning.

## Technologies Used
Python
Scikit-learn
Pandas

## Installation
pip install pandas
pip install scikit-learn

## How to Run
Run the python file:
python main.py

## Input
hours studied
attendance
previous marks

## Output
predicted marks
